The Report Card Management System is a web-based application developed using Core Python, Pandas, and Streamlit to simplify student record management and report card generation. The application provides an intuitive interface for entering student information and subject marks, storing records in memory using Pandas DataFrames and Streamlit Session State, and performing various academic calculations.

Features
Add and manage multiple student records.
Store student details including name, roll number, and marks in five subjects.
Calculate total marks and overall percentage for any student.
Automatically assign grades (A–F) based on subject-wise marks.
Display subject-wise average marks across all students.
Identify the highest scorer in each subject.
Generate a complete report card showing student details, marks, total, percentage, and grades.
Interactive and user-friendly interface built with Streamlit.
Technologies Used
Core Python – Application logic and control flow.
Pandas – Data storage, manipulation, and statistical calculations.
Streamlit – Interactive web interface for seamless user interaction.
NumPy – Imported for numerical operations and future scalability.
Project Objective

The primary goal of this project is to automate the process of student report card generation while demonstrating the use of Python for data processing, data analysis, and web application development. It eliminates manual calculations, provides quick academic insights, and offers an easy-to-use interface for managing student performance.

This project is suitable for beginners learning Python, Pandas, and Streamlit, showcasing how these technologies can be combined to build a practical educational management application.
